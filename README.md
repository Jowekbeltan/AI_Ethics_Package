# AI Ethics Assignment – Complete Package

This repository contains all materials for the **AI Ethics Assignment**:

## Contents
- `answers.pdf` — Full written answers (Part 1–4 + Bonus Task)
- `audit_notebook.ipynb` — Jupyter Notebook with fairness audit using AIF360
- `README.md` — Project overview

## How to Use
1. Open `answers.pdf` to view all written responses.
2. Run `audit_notebook.ipynb` in Jupyter Notebook.
3. Install AIF360 before running the notebook:
   ```
   pip install aif360
   ```
